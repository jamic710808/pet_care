# 泡泡爪 Pet Spa - 開發紀錄 (2026/05/12)

## 📋 今日開發進度
今日核心重點在於系統的「安全性強化」與「自動化部署優化」，確保管理員後台不會被外部人員隨意存取，並完善了本地開發與雲端環境的同步。

## 🛠 修改與新增項目

### 1. 管理員身份驗證系統 (Admin Auth)
- **登入機制**: 實作了基於 JWT (JSON Web Token) 的身份驗證，帳號密碼驗證通過後會存入 HttpOnly Cookie，有效防禦 XSS 攻擊。
- **密碼安全**: 引入 `bcryptjs` 對密碼進行加鹽雜湊 (Salted Hashing)，資料庫中僅存儲加密後的字串。
- **後台功能**:
    - **修改密碼**: 管理員登入後可直接在介面修改密碼，系統會自動同步至 Neon 雲端資料庫。
    - **登出功能**: 清除瀏覽器認證憑證並重定向至登入頁。
- **頁面保護**: 實作了全域 Middleware，所有訪問 `/admin` 的請求若未經過身份驗證，將被自動引導至 `/admin/login`。

### 2. 臺灣手機格式支援優化
- **預約系統**: 將預約表單的電話欄位預設格式從中國格式修改為**臺灣手機模式 (09XXX-XXX-XXX)**。
- **前端驗證**: 加入了 Regex 正規表達式驗證，確保客戶輸入的必定是正確的十碼臺灣手機號碼。

### 3. Vercel 部署架構修正
- **分支同步**: 解決了 `master` 分支被視為預覽分支的問題，將 Vercel 的正式發佈分支 (Production Branch) 統一指定為 `master`。
- **環境變數同步**: 解決了本地開發環境找不到 `POSTGRES_URL` 的問題，實現了本地開發與雲端 Neon 資料庫的完美連結。

## 📝 關鍵配置與依賴更新
- **新依賴**: `bcryptjs`, `jose`, `dotenv` (本地環境變數讀取)。
- **環境變數**: 雲端與本地均需配置 `POSTGRES_URL` 與 `JWT_SECRET`。
- **資料庫**: 新增 `public.admin_users` 表，用於存儲管理員資訊。
